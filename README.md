# 🚀 AI-Powered Parametric Income Insurance for Gig Workers

## 📌 Overview

This project presents an **AI-enabled parametric insurance platform** designed to protect gig delivery workers (such as Swiggy/Zomato partners) from **income loss caused by external disruptions** like extreme weather, pollution, and local restrictions.

The system ensures **automatic payouts without manual claims**, using real-time data triggers and AI-based risk assessment.

---

## ❗ Problem Statement

Gig delivery workers rely on daily earnings but face frequent disruptions such as:

* 🌧️ Heavy rainfall
* 🌫️ High pollution levels
* 🔥 Extreme heat
* 🚫 Curfews and local restrictions

### 👉 Impact:

* 20–30% loss in weekly income
* No financial safety net
* No existing income-based insurance solution

---

## 🎯 Our Solution

We propose a **Weekly Parametric Insurance Model** that:

* Continuously monitors environmental conditions
* Uses AI to calculate worker risk profiles
* Automatically triggers payouts when thresholds are exceeded
* Eliminates the need for manual claim filing

---

## 👤 Target Persona

**Ravi Kumar – Delivery Partner (Swiggy/Zomato)**

* Daily Income: ₹600
* Weekly Income: ₹4200

### 📉 Scenario:

Heavy rainfall for 2 days → ₹1200 income loss

✅ System detects rainfall threshold breach
✅ Automatically triggers payout
✅ ₹800 credited instantly

---

## 🔄 System Workflow

1. User signs up with location and work details
2. AI calculates **Risk Score**
3. User selects a **Weekly Insurance Plan**
4. System monitors:

   * Weather data
   * AQI levels
   * External disruptions
5. If trigger occurs:

   * Claim is auto-generated
   * Instant payout is processed

---

## 🤖 AI/ML Components

### 🔹 Risk Prediction

* Predicts probability of disruptions
* Assigns a **Risk Score (0–100)**

### 🔹 Dynamic Pricing

* Adjusts premium based on risk

### 🔹 Fraud Detection

* Detects:

  * Fake location usage
  * Duplicate claims
  * Abnormal patterns
